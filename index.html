Tachyon is a fault tolerant distributed Ô¨Åle system enabling reliable file sharing at memory-speed across cluster frameworks, such as Spark and MapReduce.It offers up to 300 times higher throughput than HDFS, by leveraging lineage information and using memory aggressively. Tachyon caches working set files in memory, and enables different jobs/queries and frameworks to access cached files at memory speed. Thus, Tachyon avoids going to disk to load datasets that is frequently read.

We released Tachyon 0.2.0 on April 10th, 2013 The new version is much more stable and also features significant performance improvements.

Getting Started

Check out the quick start tutorial to get Tachyon running locally in a few minutes.

Java-like File API

Tachyon has an API similar to JAVA file, providing InputStream, OutputStream interface, and efficient support for memory mapped I/O.
